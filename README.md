# Studio by HarnessGG

Studio by HarnessGG is a local desktop app for AI-assisted video editing with Codex and HarnessGG toolchains.

## How to use

> [!WARNING]
> You need to have Codex CLI, Kdenlive, FFmpeg, Python, and `harnessgg-kdenlive` installed for full local editing workflows.

```bash
bun run dev:desktop
```

For packaged builds, use the desktop artifact scripts in the root `package.json`.
